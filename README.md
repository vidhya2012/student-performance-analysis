# Student Performance Analysis (Python)

This project analyzes student marks using Python.

## Features
- Calculates average marks
- Assigns grades
- Counts pass and fail students
- Finds the topper
- Displays output in a table format

## Skills Used
- Python
- Loops
- Functions
- Dictionaries
- Conditional Statements

## Output
Displays student name, average, grade, pass/fail count, and topper.
